# Vxalware CC
**Vxalware CC** is a feature-rich BedWars script hub designed with performance, stability, and customization in mind. It provides a streamlined experience with optimized module configs, structured configuration management, and a refined user interface.

## 🔗 Links
* **Discord:** [https://dsc.gg/VxalwareCC](https://dsc.gg/VxalwareCC)
* **YouTube:** [https://www.youtube.com/@OfficialStarGazerPro](https://www.youtube.com/@OfficialStarGazerPro)

## 🚀 Usage
1. Download your preferred scripting utility.
2. Execute the following loader:

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/vxalware-bedwars-owner/Vxalware/refs/heads/main/MainScript.lua",true))()
```
